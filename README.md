# 🎵 Music Web Player

Simple Apple Music–style web app using HTML, CSS, JS.

---

# 🎧 Features

- Artists system
- Album pages
- Tracklists
- Auto-play next song
- Search songs
- GitHub Pages support

---

# ⚠️ IMPORTANT RULES

Your current system depends on EXACT file names:

Example:
- `6. Ghost Town.mp3` MUST be written EXACTLY like that in app.js

If not:
- song will not play
- no error will show unless console is open

# 🚨 COMMON ISSUES

❌ Song not playing  
→ filename mismatch OR wrong folder

❌ Artist missing  
→ JS crash (check console)

❌ Tracklist missing  
→ showAlbum() not running due to error

---

# 🚀 Recommended Fix

Rename files to remove:
- numbers
- punctuation inconsistencies
- duplicates
